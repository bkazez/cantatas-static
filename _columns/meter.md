Type all time signatures in the movement, ignoring duplicates. Use a slash (`/`) to separate numerator and denominator. For slashed C, 2, 3, etc., use a backslash (`\`).

If the score says… | …then in **meter**, type:
**4/4** | `4/4`
**C** | `C`
**3** | `3`
**₵** | `\C`

Example:
```
\C; 3/2
```
